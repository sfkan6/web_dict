## Dictionary

<hr>

### Тестовое Задание
Словарь с функциями добавления и удаления слов

Главная страница
* /dict

У вас должен быть установлен java 8+ <14
### Запуск:

1. Клонировать репозиторий и перейти в него:
```sh
git clone https://github.com/sfkan6/web_dict.git

cd web_dict
```
2. Запустить:
```sh
sbt/sbt fgRun
```
